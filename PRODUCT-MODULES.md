# 智警财安官网 — 产品模块与交互说明（企业级扩展）

本文档对应「产品特色区 / 核心功能区 / 场景展示区 / 功能演示区 / 优势对比区」及通用弹窗，说明各模块价值、适用场景与操作逻辑。

---

## 1. 产品特色专区（`#product-highlights`）

**定位**：集中呈现「AI 财务风险预警」差异化能力，与顶部 KPI 条、Hero 文案一致。

**核心价值**

- 用可量化指标（98.7% 准确率、500+ 客户、0.5h vs 24h 响应对比等）建立信任。
- 四大特色卡片覆盖：多维度风控、实时预警、定制交付、全链路闭环。
- 六维标签（资金流 / 税务 / 供应链 / 投融资 / 合规 / 舆情）说明模型覆盖域。

**适用场景**：首次访问、路演材料、投资者/管理层快速扫读。

**交互**：卡片渐入动画（CSS）；指标数字在区块进入视口时与 KPI 条同步滚动（`IntersectionObserver`）。

## 2. 企业级核心功能矩阵（`#features`）

**定位**：由 4 项扩展为 8 项，区分基础 / 高级能力，并支持行业向筛选。

**功能与价值简述**

| 类型 | 功能 | 核心价值 |
|------|------|----------|
| 基础 | 实时数据监测 | 多源接入（金蝶/用友/银行/税务示意），统一数据湖 |
| 基础 | 多等级异常预警 | 黄/橙/红与多渠道触达 |
| 基础 | 个性化评估模型 | 调研→训练→上线闭环 |
| 基础 | 智能应对策略 | 预案与审批联动，演示型前后对比数据 |
| 高级 | 财务舆情分析 | NLP + 内外交叉验证 |
| 高级 | 合规审计报告 | 监管模板组装与留痕 |
| 高级 | 跨企业对比分析 | 同业/同规模对标 |
| 高级 | 智能决策辅助 | 建议书导出与水印审计 |

**操作逻辑**

1. **筛选**：工具栏四个 Chip —「全部 / 基础功能 / 高级功能 / 行业功能」。行业功能展示 `data-industry="true"` 的卡片（与基础、高级有交集）。
2. **展开详情**：每张卡片「展开详情」切换详情区；再次点击变为「收起」（文案随语言包更新）。
3. **骨架屏**：页面加载后约 0.5s 结束骨架动画，避免空白闪烁。
4. **申请功能演示**：底部按钮打开通用弹窗，`modal_type=booking`。

**键盘**：Tab 聚焦到「展开详情」按钮后，Enter/空格触发点击（原生 button）。

## 3. 行业场景化方案（`#scenarios`）

**定位**：中小企业 / 大型集团 / 金融机构三类「痛点 → 方案 → 收益」。

**交互**

- 桌面：点击下方圆点切换场景。
- 移动：在场景区域左右滑动（`touchstart` / `touchend` 识别滑动方向）。
- 轨道使用 `transform: translateX`，随窗口 `resize` 重算位移。

---

## 4. 产品功能演示（`#product-demo`）

**定位**：模拟「数据接入 → 风险监测 → 预警推送 → 策略生成」四步，支持培训与售前讲解。

**模式**

- **分步演示**：手动「上一步 / 下一步」；在最后一步点「下一步」或自动播放结束后弹出「申请免费试用」弹窗（`modal_type=trial`）。
- **自动演示**：每约 2.2s 前进一步，第四步结束后同样弹出试用弹窗。
- **暂停**：清除自动定时器。
- **重播**：回到第 1 步并刷新旁白。

**辅助**

- **复制演示链接**：写入剪贴板（失败时仍 Toast 提示为模拟成功，视浏览器权限而定）。
- **下载演示说明 PDF**：前端模拟 Toast，不生成真实文件。

**语言切换**：隐藏域 `demo-state-step` 记录当前步骤，`applyLanguage` 后按步骤刷新旁白文案。

## 5. 产品优势对比（`#comparison`）

**定位**：技术、准确率、响应、功能、服务、成本等维度一行对比，突出我方列（样式上使用绿色强调）。

**竞品名称**

- 勾选「隐藏竞品名称」：表头显示「友商 A / 友商 B」（中文）或对应英文占位。
- 取消勾选：显示完整示意名称（如 CloudRisk Pro / FinGuard Suite），便于内部分析时脱敏或展示。

**定制化方案对比**：按钮打开弹窗，`modal_type=compare`。

---

## 6. 通用弹窗（`#modal-backdrop`）

**行为**

- 打开：设置标题、说明与 `modal_type`（booking / trial / compare）。
- 关闭：关闭按钮、点击遮罩、`Escape`。
- 提交：前端模拟成功 Toast 并关闭（无后端）。
